AIDT (AI 디지털 교과서) 수학, 영어

개발 기간
2023-09 ~ 진행중
담당 업무
  • 문항 뷰어 개발
  • 여러 유형의 문항이 들어가는 문항 뷰어 개발 참여 (수업 화면 담당)
  • 수업 전반적인 교사 <-> 학생 간의 양방향 소통 위주 개발 (소켓 메세지 사용)
  • 학생모으기, 질문하기, 북마크, 활동하기, 필기 모드, 문항 채점, 문항 풀이 초기화, 스크롤 모드/슬라이드 모드, 다른 문제 풀기 기능, 이북 뷰어 연동
  • 수업 도구 개발
  • 수업 보조도구(화면고정 툴) 기능 개발
  • 출석부, 주목 시키기, 판서(fabric js), 교과 도구, 타이머/스톱워치, 발표자/모둠 뽑기, 북마크, 즉석 퀴즈, 화면 확대/축소, 툴 편집
  • 기존의 Allvia 서비스에서 사용하던 수업 도구 프로젝트를 가져와 AIDT서비스에 맞게 리팩토링을 진행했습니다.
    • AIDT서비스에서 수업 도구는 store의 역할이 축소되었고, 추후 유지보수를 고려하여 Redux -> Mobx 로 변환하였습니다.
    • React를 18버전으로 업데이트하여 class형 함수를 전부 Hook으로 리팩토링을 진행했습니다.
    • typescript를 제대로 활용하기 위해 전체 프로젝트 내에 누락된 type을 추가하였습니다.
    • gitlab의 Merge Request 기능을 이용하여 code review를 도입했습니다.
